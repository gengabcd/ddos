## Usage
syntax: ./flood [[-t IP] [-p PORT] [-r]

options:
	-t the target: IP (default 127.0.0.1)
	-p the target: PORT (default 80)
	-r set RST flag

	Example: sudo ./flood -t 192.168.0.1 -p 8080"
